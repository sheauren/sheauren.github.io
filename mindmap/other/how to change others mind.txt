* Reactance
    * 當警語變成推薦
        * 別做->更想做
    * 人需要自由與自主權
        * 心情愉悅與長壽
        * 自行決定就算更痛苦也還是希望保持自主權
    * 抗拒心理&反說服雷達
        * 失去自主權->重新掌握自主權->去做被禁止的事情
        * 限制會帶來抗拒
        * 無人勸說的時候->做自己想做的事情
        * 公司希望員工說出心聲->公司叫妳這麼做!=自己想這麼做
        * 發生抗拒心理-> 迴避/不理會/唱反調/質疑內容
    * 解決案例: 年輕人戒菸 -> 停止告訴你該做什麼
        * 招開青少年菸草高峰會
            * 與學生討論了解問題
            * 列出產業事實
                * 操弄風向推銷香菸
                * 企業控制政治系統
                * 體育、電影、電視讓抽菸看起來很cool
            * 讓學生想希望怎麼做
                * 全州性新組織： 學生反菸會
                    * 作業練習簿+菸草資訊利潤
                    * 真相廣告：做出版就是要賺錢->菸草廣告
                * 全州性->全國性->菸草公司打官司要求終止此計畫
        * 了解真相讓年輕人自己選擇
            * 讓自己積極參與
            * 不是被動的旁觀者
            * 不直接下決定也不完全靜觀其變
            * 找出中庸之道引導對方
    * 解決手法：
        * 提供選單: 引導式選擇
            * (X)強迫吃什麼食物
            * (O)你要先吃什麼食物
                * 沒有被強迫是我自己挑想吃的
        * 循循善誘: 發問但不直接發表定論
            * (X)憑什麼告訴我要怎麼做
            * (O)詢問你如何做到你要的目標
                * 提供客觀數據讓執行者知道目標的難度與落差
                * 讓執行者評估執行難度與預估工時
                    * 提供客觀數據當建議
            * 類似提供選單，也當聽眾不急著反駁，找出問題的感受，讓人樂於談論自己跟找出答案
            * 問題的答案是他自己的答案不是隨隨便便的答案
                * 按結論去做
            * 問題要選好，讓回答者慢慢往目標移動
        * 突顯差異: 指出言行不一之處
            * 抽菸的孩子=>孩子借火被拒絕之後的卡片："你關心我，為什麼你不關心你自己"
            * 抽菸者建議其他人做的是，自己卻沒做
            * 人會尋求內在一致性，態度理念行為一致，避免心虛
            * 態度行為的差距
                * 省水海報簽名：如果可以，我也做得到
            * 說服別人放棄->改變參照點
                * 收掉賠錢部門->再給機會?在多給時間? 無法放手      
                * 從零開始或換個執行長會建議留這部門嗎? 砍
            
        * 從理解著手
            * 不要開頭就說自己想要什麼，先去認識想要改變的那個人
            * 策略性同理心: 了解根本問題出在哪
                * 為什麼苦惱
                * 需要什麼
                * 建立連結與奠定影響力
                * 感到關懷與理解
            * 你和我會一起想出辦法，不讓結果超出你我的掌控
            * 讓那個人覺得是自已的點子
                * 不是命令而是為他著想去認為達成目的最好的方法是我期待的結果
    * 轉換抗拒心理
        * 從逼迫與試圖說服 => 鼓勵對方說服自己
        * 選項式或者引導式 => 讓人選擇自己的道路
        * 建議他人做的事情 => 本人做的事情        
* Endowment
    * 現況誤差
        * 以現在狀態去評估而非客觀評估
        * 如果避免此狀況?
            * 不採取行動的成本現形
                * 不行動的損失=>持續擴大的問題不會終止=>不行動遲早會關門
                * 大毛病會願意治療，輕微問題反而不處理
                * 維持現狀非毫無成本，仍然要付出代價
                    * 儲蓄而非投資的無形損失
                * 改變成本效益的時間差很長，缺乏耐心之下造成遲遲不肯改變
                    * A是A+的敵人，固守現狀
                    * 吃甜點以後會變胖，但是現在得放棄眼前的甜點
                * 催化劑的作用
                    * 讓現況成本現形，方便看出目前與可以做到的差異
                    * 不是看出新舊的效益，而是突顯不變的無形損失
            * 破釜沉舟
                * 斷後路只能勇往直前
                * 更新電腦的成本
                    * 舊電腦直接無法使用、斷網
                * 汽車舊零件產能下降、提高售價
            * 減輕敝掃自珍
                * 催化劑除了讓你對新事物放心，也要讓你放下舊事物
                * 脫歐的行動
                    * 口號: take back control
                        * 不是改變是拿回原本失去的
                    * 宣傳： 每周給歐盟的錢xxx不如給健保局
                * 川普: 讓美國再次偉大
                * 不強調創新，強調歸回基本能力
    * 損失規避 loss aversion
        * 改變的優缺點，每個人心中比重不同
            * 期望值*發生率 : 潛在獲利為潛在損失的2.6倍才願意下注
            * 新東西的好處只要2倍，好處必須是在意的點
                * 好處 > switching cost            
* Distance
    * 跨越鴻溝
        * 同溫層現象
            * 接觸反對方資訊後，立場更強烈的反對
                * 文章並非勸說造成的反效果
                * 單純提供資訊仍然造成強烈反對
        * 導正錯誤信念
            * 直覺數據能讓人跟上最新思維 => 實際並非如此
            * 會用懷疑的眼光看待數據 => 增強誤解
            * 改變看法，強烈訴求效果會變差，容易掉到拒絕區
                * 兩造極端以外的中間區間的人需要提供資訊
                    * 接受區的資訊能有幫助
                    * 拒絕區的資訊會造成反效果
                    * 拿原觀點去比較接收到的資訊
                    * 溫和資訊比強烈訴求往目標前進3倍
    * 確認誤差
        * 人習慣去尋好與吸收符合原有觀點的資訊
            * 拿現有觀點比較
        * 檢視特定信念相關的證據，只看預期想看的
            * 預期答案=> 我可以相信這事情嗎
            * 非預期答案 => 我一定得相信這個嗎
                * 難以改弦易轍
        * 催化劑如何避開拒絕區? 減少距離三方法
            * 找出有爭取空間的中間派 => 不要試圖改變每個人
                * DM、廣告於初選無效=>站在同一邊，普選時站在兩邊
                * 特定政黨支持者=>換邊困難
                * 高度關切議題及領域者=>非黑即白
                * 不在意者=>接受區很廣
                * 找出關心某些議題搖擺的選民
                    * 不廣發此議題給所有選民=>送到極端反對者身上=>損己利人
                    * 有極端感受的中間派=>轉換陣營大
                * 找出意見矛盾或有意改變的行為痕跡
                * 無現成數據則從測試中學習
                    * 找出在該小群體有效的方法
            * 從「小忙」問起 => 鋪踏腳石
                * 一步到位會被拒絕(拒絕區)
                    * 先讓對方答應無關緊要的要求(非拒絕區)，隔幾天在詢問上述問題(同意機率提升)
                    * 小忙會移動接受區
                * 一天喝太多汽水3L(X) => 先試著一天喝2L汽水就好 => 1L => 0L (O)
                * 循序漸進 : 改變他人不要一開始野心太大
            * 改變場上位置，找出共同點
                * 找出沒有分歧已有共識面相，把該面相當轉向支點
                    * 不硬闖阻塞的道路，從支點繞過去
                * 深度遊說 (傳統遊說: 丟下資訊)
                    * 開誠布公的互動，討論複雜的議題
                    * 讓對方安心的說出真實感受，包含謾罵
                * 傳統的將心比心，從別人觀點出發並不實際
                    * 鼓勵從自身經驗找出類似情境，只是回想有類似感受            
* Uncertainty
    * 改變不確定稅(Uncertainty Tax)
        * 人會規避風險，不確定因素降低意願，就算客觀因素比較優
        * 人是真心厭惡不確定性，不確定性比負面結果還遭
        * 不確定性會降低換個方式做的價值
    * 按下pause
        * 考試結果未知可能要補考時，即期優惠旅遊方案選擇三選一：購買/不買/150元保留費
            * 已知考試結果: 購買/不買 居多
            * 未知考試結果: 150元保留費 居多
    * 可實驗性
        * 創新的擴散(diffusion of Innovations)：靠5種特性來解釋
            * 可試性 (trialability)
                * 可測試、可體驗
                * 越容易試用越快採用
                * 減少適用障礙 (協助發現->降低前期成本->讓事情變的可逆)
                    * 免費增值(freemium)
                        * 免費使用
                        * 買單轉換成本
                    * 降低前期成本
                        * 免運費 => 消除不確定性障礙 => 試用                    
                        * 新科技試用、新車試駕 
                        * 省下180運費的價值達於商品降價300，不是因為價格而是因為不確定性                    
                    * 讓人有機會認識
                        * 與飯店合作免費搭車服務，增加體驗機會，增加潛在客戶
                        * 超市免費試吃
                        * 訂閱制兒童教材鼓勵辦慶生會並提供品牌禮物
                    * 後悔也沒關係
                        * 讓決定變成可逆
                        * 怕沒準備好養寵物，提供兩周試養期
                        * 退貨限制少的產品反而有更高的利潤
                            * 口碑銷售量皆提升
                * 利用慣性優勢
                    * 試用讓買家心態從取得變成留住
                        * 不願意花5元去取得
                        * 願意花5元避免失去
                    * 越容易試用買的機會越高
                        * 恐新症
                        * 利用社交連結試用
* Corroborating Evidence
    * 小石頭與大石頭
        * 強態度(strong attitude)
            * 難以改變
            * 強大證據才能撼動
            * 爭議大、風險高、耗時長、貴
        * 弱態度(weak attitude)
            * 較為無感，改變難度較輕
            * 小量證據就能改變
    * 推薦
        * 被推薦事務+推薦人自身+契合度 => 是否搬動石頭
    * 眾口一致 => 證明
    * 證據 = Who When How
        * Who: 找誰 or 證據提供者是誰
            * 旁邊笑的人跟你類似 => 跟著笑
            * 旁邊笑的人與你對立 => 不會笑
            * 相似度決定判斷搬動石頭的程度
        * When: 佐證時機
            * 何時揭露影響力最大
            * 同事推薦影片，隔天另一個同事也推薦 => 好奇
            * 同事推薦影片，隔三周之後另一個同事也推薦 => 無感
                * 間隔太長會削弱群體影響力            
        * How: 如何分配，依照門檻而定
            * 灑水器策略: 雨露均霑，到處都灑一點，都照顧到
                * 小石頭(O)
                * 大石頭(X)
            * 消防水帶策略: 集中灌注，濕透在換下一區
                * 大石頭(OK)

    * 推動吃內臟的手法
        * 降低料理門檻=>把內臟偷偷混入食材
        * 不願意改變的損失=>損害國軍戰力
        * 小組討論取代演講=>讓家庭主婦討論如何戰勝烹飪障礙

    * 催化劑的力量
        * 看法是有可能改變的
        * 重大轉變需要花很長時間
        * 解決路障可以促進改變
